# Prologku
A Sudoku validator/solver in Prolog

## Examples
In your Prolog REPL, you can validate Sudoku solutions:
```prolog
?- sudoku([
[1, 2, 3, 4, 5, 6, 7, 8, 9],
[4, 5, 6, 7, 8, 9, 1, 2, 3],
[7, 8, 9, 1, 2, 3, 4, 5, 6],
[2, 1, 4, 3, 6, 5, 8, 9, 7],
[3, 6, 5, 8, 9, 7, 2, 1, 4],
[8, 9, 7, 2, 1, 4, 3, 6, 5],
[5, 3, 1, 6, 4, 2, 9, 7, 8],
[6, 4, 2, 9, 7, 8, 5, 3, 1],
[9, 7, 8, 5, 3, 1, 6, 4, 2]]).
true.
```

You can even give a partial solution and let Prolog work out the rest:
```prolog
?- solve([
[4,_,_,7,_,1,6,_,8],
[6,_,_,_,_,_,1,_,_],
[_,_,_,_,_,_,5,4,_],
[_,_,3,_,_,6,_,_,_],
[_,_,9,_,7,_,_,_,3],
```
